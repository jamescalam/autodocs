    Initialise DocsBuilder class. Checks the given documentation directory for Bootstrap templates, if not found will download from here. Also compiles useful Regular Expressions (TK add class finder to it's own function).

    docs_dir The local/global path to the documentation directory.

  • extract

    extract(code)

    Function for extracting imported libraries/modules, global variables, functions and classes.

    code String containing the Python code to be documented.

  • build

    build(path, overwrite)

    Function for building HTML docs using extracted data.

    path* String containing the local/global filepath to the Documentation directory. The HTML files will be saved here. The default is './Documentation'.
    overwrite* True/False determining whether pre-existing files will be overwritten without warning. If False then overwrite will still be possible but a warning will appear if there are any pre-existing files, which will confirm the user's intention in overwritting or otherwise.
